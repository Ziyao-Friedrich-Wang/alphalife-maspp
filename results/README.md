# Reproduced Result Snapshots

This directory contains compact CSV snapshots used to verify the paper tables.
They are derived outputs only; the raw empirical factor-return panel is not
included in this repository.

The main files are:

- `maspp_state_control_summary.csv`: main performance table.
- `maspp_state_control_bootstrap.csv`: moving-block bootstrap comparisons.
- `maspp_state_control_downside_mechanism.csv`: cluster-budget mechanism diagnostics.
- `maspp_state_control_robustness.csv`: shrinkage and cluster-count robustness.
- `maspp_state_control_subperiod.csv`: subperiod performance.
- `maspp_state_control_deployability.csv`: turnover, switching, and exposure proxies.
